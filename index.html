Objetivo del Proyecto

El presente proyecto tiene como objetivo plasmar lo aprendido en el curso de Arquitectura de Computadoras, al poner en funcionamiento la máquina de burbujas, a través de la placa de Arduino, el código, sensores y los demás componentes a usar, para lograr una adecuada ejecución.

notes

Descripción del Proyecto

La máquina de burbujas genera burbujas al presionar uno de los dos botones, el primer botón permite un funcionamiento continuo, mientras que el segundo botón realizara una única secuencia. Luego de presionar uno de los dos botones, el sensor ultrasonido verificara la distancia con el agua para las burbujas, si el agua tiene una altura adecuada los servos moverán el brazo del burbujero hasta que se remoje en el agua y se coloque a la altura del ventilador. El ventilador se encenderá por unos segundos para generar las burbujas, luego de un tiempo se apagara, finalizando la secuencia.

Para la elaboración del proyecto se tuvieron que conseguir los implementos a usar, así como el esquema de la conexión de los componentes para luego hacer el armado y la prueba de funcionamiento con el código que subimos a nuestro Arduino.

//Burbujero.ino
#include <Servo.h>
#include "Notas.h"
Servo servoHorizontal, servoVertical;
int anguloVerticalArriba = 90;
int anguloVerticalAbajo =20;
int anguloVerticalVentilador = 44;
int anguloHorizontalOrigen = 0;
int anguloHorizontalFinal = 70;
int boton1uso, botonContinuo;
int pinBoton1Uso = 2;
int pinBotonContinuo = 4;
int pinRele = 11;
int triggerPin = 5;
int echoPin = 6;
int nivelAguaOK = 8;//cm
int distanciaActual = 0;
int pinVerde = 8;
int pinRojo = 3;
int speakerPin = 12;
void setup() {
//Configuracion se los pin de los servo.
servoHorizontal.attach(9);
servoVertical.attach(10);
// Se abre el monitor serie.
Serial.begin(9600);
//Configuracion de los pin de los botones
pinMode(pinBoton1Uso, INPUT);
pinMode(pinBotonContinuo, INPUT);
pinMode(pinRele, OUTPUT);
pinMode(LED_BUILTIN, OUTPUT);
pinMode(triggerPin,OUTPUT);
pinMode(echoPin,INPUT);
pinMode(pinVerde,OUTPUT);
pinMode(pinRojo,OUTPUT);
pinMode(speakerPin, OUTPUT);
retornoPosicionInicial();
}
void loop() {
boton1uso = digitalRead(pinBoton1Uso);
botonContinuo = digitalRead(pinBotonContinuo);
distanciaActual = comprobarRecipiente();
Serial.print("Distancia: ");
Serial.println(distanciaActual);
if(distanciaActual < nivelAguaOK){
//Notificamos el nivel de líquido válido con el led.
digitalWrite(pinVerde, HIGH);
digitalWrite(pinRojo, LOW);
Serial.print("Distancia Correcta: ");
Serial.println(distanciaActual);
if(boton1uso == HIGH){
ejecutarCiclo();
}
else if(botonContinuo == HIGH){
//NO-NADA
tone(speakerPin, NOTE_G4);
delay(1000);
noTone(speakerPin);
delay(1000);
botonContinuo = digitalRead(pinBotonContinuo);
while(botonContinuo == LOW){
ejecutarCiclo();
digitalWrite(pinRojo, HIGH);
delay(2000);
botonContinuo = digitalRead(pinBotonContinuo);
digitalWrite(pinRojo, LOW);
delay(2000);
}
}
}else{
//Tocar la campana
digitalWrite(pinVerde, LOW);
digitalWrite(pinRojo, HIGH);
}
}
void ejecutarCiclo(){
//Encendemos el led para indicar que se ejecuta el ciclo
digitalWrite(LED_BUILTIN, HIGH);
//Mover servo vertical
servoVertical.write(anguloVerticalAbajo); //Abajo
delay(2000);
servoVertical.write(anguloVerticalArriba); //Arriba
delay(1000);
//Mover servo horizontal
servoHorizontal.write(anguloHorizontalFinal); //Final
delay(1000);
//Lo volvemos a bajar para el ventilador
servoVertical.write(anguloVerticalVentilador);
//Iniciar ventilador
delay(1000);
digitalWrite(pinRele, HIGH);
//Para ventilador
//delay(10000);
GameOfThrones();
digitalWrite(pinRele, LOW);
//Melodía

//Retorna vertical
servoVertical.write(anguloVerticalArriba); //Arriba
delay(1000);
//Retorna horizontal
servoHorizontal.write(anguloHorizontalOrigen); //Final
delay(1500);
//Apagamos el led para indicar el fin de ciclo
digitalWrite(LED_BUILTIN, LOW);
}
void retornoPosicionInicial(){
servoVertical.write(anguloVerticalArriba);
servoHorizontal.write(anguloHorizontalOrigen);
digitalWrite(pinRele, LOW);
digitalWrite(LED_BUILTIN, LOW);
digitalWrite(triggerPin, LOW);
//pulseIn(EchoPin, LOW);
digitalWrite(pinVerde, LOW);
digitalWrite(pinRojo, LOW);
}

int comprobarRecipiente() {
long duracion, distanciaCm;
//Lanzamos un pulso
digitalWrite(triggerPin, LOW);
delayMicroseconds(4);
digitalWrite(triggerPin, HIGH);
delayMicroseconds(10);
digitalWrite(triggerPin, LOW);

//Medimos el tiempo de respuesta en
microsegundos
duracion = pulseIn(echoPin, HIGH);

//Convertimos a distancia, en cm **
Distancia=(Tiempo/Velocidad)/2 **
distanciaCm = duracion * 10 / 292/ 2;
//El sonido tarda 29,2 microsegundos en recorrer un centímetro.
return distanciaCm;
}

Funcionamiento

La máquina de hacer pompas tiene dos modos de funcionamiento. Con un botón se conseguirá una única ejecución y con la otra es conseguirá que el brazo esté continuamente cargando el pompero y haciendo pompas hasta que sea detenido, para lo que habrá que pulsar de nuevo dicho botón cuando se encienda el led rojo al final de carrera.

Antes de permitir iniciar la ejecución, el sensor de ultrasonidos se asegura de que el recipiente esté cargado y, si no es así, mantiene encendido el led rojo indicando que no se puede arrancar. Así mismo no atenderá el pulsado de los botones.

Si pulsamos el botón de ejecución en bucle, sonará un pitido confirmándonos la selección. A continuación, el brazo bajará el pompero, introduciéndolo en el recipiente y lo volverá a levantar cargado. Una vez esté arriba se desplazará lateralmente hasta ponerlo a la altura del ventilador y volverá a bajarlo para que quede frente a éste.

El ventilador se encenderá y sonará la melodía de Juego de Tronos mientras divertidas pompas de jabón surgen para regocijo de todos los presentes. Al finalizar la melodía, el ventilador se detendrá, el brazo subirá y retornará a la posición original.

En este momento, se encenderá durante un breve periodo el led rojo a la vez que el led verde. Será el momento, antes de que vuelva a apagarse, en el que tendremos que volver a pulsar el botón de ejecución continua si queremos que se detenga. De no ser así, pasados 2 segundos se volverá a iniciar otro ciclo.

Conclusiones y recomendaciones

Podemos concluir que el proyecto realizado nos muestra la versatilidad de la placa Arduino, que al ser una plataforma de código abierto esta permite hacer modificaciones a gusto del desarrollador. Es por ello que, en primera instancia, el proyecto no podría funcionar como uno lo espera y se deberán hacer ciertas modificaciones para su perfecta implementación.

Es recomendable que, para una mayor fluidez, se tenga un conocimiento básico de algunos componentes electrónicos, así como del lenguaje C++ que es el que usa Arduino como base. También recomendamos agregar un mejor buzzer para obtener un sonido mas definido. En próximos proyectos se podría integrar motores paso a paso, e incorporar un burbujero con mas orificios, de esta manera se podrían generar una mayor cantidad de burbujas por ciclo de funcionamiento.
